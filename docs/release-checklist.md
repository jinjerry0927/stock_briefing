# Release Checklist

## 코드
- `npm run lint` 통과
- `npm run build` 통과
- `git diff --check` 통과
- 불필요한 샘플 자산과 생성 산출물이 추적되지 않는지 확인

## 보안
- `.env.local`이 추적되지 않는지 확인
- `data/*.db*`가 추적되지 않는지 확인
- README, docs, 코드 주석에 실제 API 키나 개인 보유 종목 메모가 없는지 확인
- 공개 저장소에 올릴 수 없는 파일은 `.gitignore`에 포함

## 제품
- 관심종목 추가/삭제/검색 확인
- 보유종목 추가/편집/삭제 확인
- USD/KRW 환율 fallback 확인
- 보유/관심 차트의 1개월, 3개월, 6개월, 1년 범위 확인
- 뉴스 링크가 새 창에서 열리는지 확인

## 문서
- README가 현재 기능과 실행 방법을 설명하는지 확인
- 데이터 소스 한계와 투자 조언이 아니라는 점을 명시
- 공개 전 보안 기준을 문서에 남김
